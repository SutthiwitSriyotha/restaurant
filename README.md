# Restaurant Management System (Project สำหรับฝึกฝนทักษะ)

โปรเจกต์นี้พัฒนาขึ้นเพื่อเป็นส่วนหนึ่งของการเรียนรู้และฝึกฝนทักษะการพัฒนา Full-stack Web Application โดยเน้นที่การสร้างระบบจัดการร้านอาหารที่ใช้งานได้จริง ตั้งแต่การสั่งอาหารผ่าน QR Code ไปจนถึงการจัดการหลังบ้าน

## 🚀 เทคโนโลยีที่ใช้ (Tech Stack)

โปรเจกต์นี้เลือกใช้เทคโนโลยีสมัยใหม่เพื่อให้ระบบมีความรวดเร็วและรองรับการขยายตัว:

* **Frontend:** [React](https://reactjs.org/), [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/), [Material UI](https://mui.com/material-ui/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Database:** [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)
* **UI Framework:** [Vuexy](https://pixinvent.com/vuexy-nextjs-admin-template/) (สำหรับส่วน Admin/Dashboard)

## ✨ คุณสมบัติของระบบ (Features)

ระบบถูกออกแบบมาให้ครอบคลุมการทำงานหลักของร้านอาหาร ดังนี้:

### 📱 สำหรับลูกค้า (Customer Side)
* **QR Code Ordering:** ลูกค้าสามารถสแกน QR Code ประจำโต๊ะเพื่อเข้าถึงเมนูได้ทันที
* **Digital Menu:** แสดงรายการอาหารแยกตามหมวดหมู่ พร้อมรูปภาพและราคาที่ชัดเจน
* **Cart Management:** เลือกอาหาร ปรับแต่งจำนวน และตรวจสอบรายการก่อนสั่ง
* **Real-time Order Tracking:** ติดตามสถานะของอาหารที่สั่งได้แบบ Real-time (เช่น กำลังปรุง, พร้อมเสิร์ฟ)

### 👨‍💼 สำหรับผู้ดูแลและพนักงาน (Admin & Staff Side)
* **Dashboard:** สรุปภาพรวมยอดขายและสถิติการสั่งอาหารที่น่าสนใจ
* **Order Management:** ระบบจัดการรายการสั่งอาหารจากลูกค้า แบ่งสถานะการทำงานชัดเจน
* **Menu Management (CRUD):** เพิ่ม ลบ แก้ไข ข้อมูลเมนูอาหาร หมวดหมู่ และราคา
* **Table Management:** จัดการสถานะของโต๊ะอาหาร (ว่าง, กำลังใช้งาน, รอเช็คบิล)
* **Inventory & Database:** เชื่อมต่อกับ MongoDB Atlas เพื่อจัดเก็บข้อมูลอย่างเป็นระบบ

## 🛠️ วัตถุประสงค์ของโปรเจกต์
1. เพื่อฝึกฝนการใช้ **Next.js** และ **TypeScript** ในการสร้าง Application ที่มีความซับซ้อน
2. เรียนรู้การจัดการ State และการรับส่งข้อมูลระหว่าง Frontend และ Backend
3. ฝึกการออกแบบ UI/UX ให้เหมาะสมกับการใช้งานจริง ทั้งบน Mobile และ Desktop
4. ศึกษาการใช้งาน NoSQL Database (MongoDB) ในการจัดการข้อมูลร้านอาหาร
